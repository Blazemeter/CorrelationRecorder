Correlation Replacements

You can select one Correlation Replacement from this ComboBox, adding configurations that will allow you replace values on each request.

Notice that you don't need to pick a C. Replacement if you don't need it since a rule is complete, as long as it has either a Correlation Replacement
or a Correlation Extractor selected.

For more information about Replacements and how to use them, check the Documentation at List of Correlation Replacements
