Function Correlation Replacement

Use the Regular Expression (RegEx) match the value that will be replaced with the Variable Name. Will apply to all the matches as long as the obtained value is stored and, Reference variable should be a function, obtaining the value of it when replaying the recording.


Field Description Default Value
RegEx Regular Expression used to do the matching param="(.+?)"
