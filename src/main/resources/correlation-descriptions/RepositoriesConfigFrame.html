Repository Manager

Here you can add both Local and External Repositories into your plugin, allowing you to load and save them whenever you need to access them.

Column's Description

This are the fields that are required for each repository


Column Description Allowed Values
ID Represents the identifier of the repository. It has to be unique Letters, numbers and special characters: "_" or "-" (without the quotes)
URL Represents the URL/Path of the repository file All but spaces

By default, the central is added to the plugin, and all the Repositories shared by the community will be automatically downloaded into your machine
when they are released in the Central Repository.

Usage Instructions

  • To add a repository: click on the "Add" button and fill the required information.
  • To delete repository: click on the row for the repository to delete, and press the "Delete" button
  • To save changes of this table, press the "Save" button. Validations will be applied at this point

NOTE: If you have any issue when saving your changes, there are two choises: you either fix the
problems or discard the changes made on the table. Saved changes wont take place until no issues are found.

For more information about repositories, how to share or create them, please check the Documentation at External Repositories
