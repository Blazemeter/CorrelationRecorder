JSON Correlation Replacement

Use the JSONPath Expression match the value that will be replaced. Will apply to all the matches as long as the obtained value is stored.


Field Description Default Value
JSON JSONPath Expression used to do the matching $.jsonpath.expression
Replacement String This field is used to set JMeter Functions/literals to be used on the comparison of the replace
match value (if value is not ignored) 		Empty
Ignore value This check will determine if the match will be compared against all the JMeter Variables or even
the execution of a function declared on the Replacement String field (if any).
If checked, the replacement will occur without comparing. 		Unchecked

JMeter use JSONPath syntax from Jayway JsonPath. Use the Jayway JsonPath syntax documentation as a reference.

For more information about Replacements and how to use them, check the Documentation at List of Correlation Replacements
