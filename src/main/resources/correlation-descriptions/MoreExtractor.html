Correlation Extractors

You can select one Correlation Extractor from this ComboBox, adding configurations that will allow you get values from each response

Notice that you don't need to pick a C. Extractor if you don't need it since a rule is complete, as long as it has either a Correlation Replacement
or a Correlation Extractor selected.

For more information about Extractors and how to use them, check the Documentation at List Correlation Extractors
